# Product Spec

## Product Identity
RadarArchive is a historical weather replay app focused on radar history.

## Core Modes
1. Live Mode — current radar, alerts, forecast.
2. History Mode — date/time selection, radar playback, warning history, layer toggles.
3. Forecast Compare Mode — later phase; compare HRRR forecast output to observed radar.

## MVP Features
- Mobile-friendly PWA
- Map view
- MRMS radar layer
- Time slider
- Play/pause
- Layer panel
- Saved default location
- Subscription-ready access rules

## Later Features
- NWS warning polygons
- WPC fronts
- HRRR wind streamlines
- HRRR simulated reflectivity
- storm reports
- radar-derived storm tracks
- event bookmarks
- export MP4/GIF
- address weather history report
